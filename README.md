# BBS-
用户相关功能
    1. 登录注册
    2. 账号绑定(QQ邮箱)
    3. 设置头像(后期考虑自定义上传头像)
    4. **消息通知(私聊窗口)
    5. **个人主页(更新资料,设置,修改密码)

帖子相关功能
     1. 发帖和编辑(支持发送图片)
     2. 点赞收藏
     3. **回复(包括楼中楼回复)
     4. 帖子分类
     5. 话题标签
     6. 置顶帖  精华帖
     7. **管理面板

身份验证  JWT

前端技术栈: vue + Ajax+BootStrap 
IDE: VSCode
 
后端技术栈: SpringBoot+Mybatis(plus)+ MySQL
IDE: IDEA
